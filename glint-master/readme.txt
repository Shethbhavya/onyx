SETUP & INSTALLATION
-------------------------------------------
1. Clone the repository:
   git clone https://github.com/Shethbhavya/onyx.git

2. Navigate to the project folder:
   cd onyx

3. (Optional) If using a backend, set up a virtual environment or install dependencies:
   - For Django: pip install -r requirements.txt
   - For Node: npm install

4. Open the website locally:
   - For static site: Open `index.html` in browser
   - For Django/React: Run your dev server
